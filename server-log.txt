📲 二维码页面: http://localhost:3000/api/qrcode

📋 API 端点:
   GET  /api/status          - 服务器状态（含移动端信息）
   GET  /api/qrcode           - 移动端访问二维码页面
   POST /api/project/save    - 保存项目
   GET  /api/project/list    - 项目列表
   POST /api/gallery/save    - 保存截图
   GET  /api/gallery/list    - 截图列表
   POST /api/share           - 生成分享链接
   GET  /api/online-music/list    - 获取在线音乐（第三方API）
   GET  /api/online-music/stream  - 代理音频流（解决跨域）

📱 PWA 支持:
   /manifest.json             - PWA 清单
   /sw.js                     - Service Worker
   /public/icons/             - 应用图标

💡 提示:
   • 手机浏览器打开地址 → 点击"添加到主屏幕" → 像APP一样使用
   • 离线也可使用基础绘图功能（Service Worker 缓存）
   • 支持 Ctrl+C 停止服务器
